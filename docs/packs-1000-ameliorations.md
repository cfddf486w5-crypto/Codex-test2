# Packs d’amélioration (1000)

Liste complète des packs activés par le bouton **Appliquer les 1000 améliorations**.
Chaque pack contient **100 améliorations** (1000 total), en mode **offline-first**, **priorité iPhone**, et **sans dépendance externe**.

1. `offline` — cache, fallback, reprise locale, continuité hors connexion.
2. `iphone` — ergonomie iOS, safe-area, lisibilité et navigation mobile.
3. `ux` — clarté des interactions, feedback, historique et guidage.
4. `performance` — exécution locale légère, batching, worker IA.
5. `fiabilite` — validations d’import, fallback, robustesse de parsing.
6. `securite` — zéro API externe, données locales, isolation runtime.
7. `data` — stockage local complet, export/import multi-store.
8. `ia` — moteurs IA embarqués (anomalie, vecteur, prédiction, simulation).
9. `github` — structure statique versionnable, compatible GitHub Pages.
10. `ops` — audit, conformité, horodatage, pilotage en un clic.

> Les packs sont aussi exposés côté code via `listAllImprovementPacks()` dans `app/improvements.js`.
