# Document de formation continue — LIA

## Objectif
Former LIA en autonomie pendant plusieurs heures pour :
- lire des documents Excel,
- écrire des informations propres,
- calculer des indicateurs,
- produire des réponses métier exploitables.

## Mode opératoire (autonome)
1. Cliquer sur **Charger 40 prompts**.
2. Cliquer sur **Formation continue**.
3. Laisser tourner le mode automatique (1 prompt toutes les 5 secondes).
4. Cliquer sur **Stop** pour arrêter.

## Boucle de formation recommandée
- **Cycle 1 (30 min)** : lecture, nettoyage, structure.
- **Cycle 2 (30 min)** : calculs, KPI, contrôles.
- **Cycle 3 (30 min)** : rédaction de réponses automatiques.
- **Cycle 4+ (illimité)** : répétition + corrections via feedback positif/négatif.

## Bonnes pratiques
- Toujours vérifier les colonnes obligatoires avant calcul.
- Normaliser les dates, nombres et unités.
- Logguer les anomalies détectées.
- Sauvegarder un backup JSON régulièrement.

## Exemples de traitements Excel attendus
- Lire un fichier et extraire les 10 premières lignes pertinentes.
- Écrire une colonne calculée (TVA, marge, score, statut).
- Calculer des totaux, moyennes, écarts et classements.
- Produire une réponse texte claire dans une colonne **Réponse IA**.
- Identifier les données manquantes et proposer une correction.

## Résultat attendu
Après plusieurs heures de formation continue, LIA doit être capable de :
- traiter rapidement un classeur Excel multi-feuilles,
- expliquer ses calculs,
- répondre de manière fiable et actionnable,
- améliorer sa qualité via le feedback utilisateur.
