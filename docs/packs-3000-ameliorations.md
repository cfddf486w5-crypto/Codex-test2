# Packs d’amélioration (3000)

Liste complète des packs activés par le bouton **Appliquer les 3000 améliorations**.
Chaque pack contient **300 améliorations** (3000 total), en mode **offline-first**, **priorité iPhone**, et **sans dépendance externe**.

## Répartition des packs

- Offline: 300
- iPhone: 300
- UX: 300
- Performance: 300
- Fiabilité: 300
- Sécurité: 300
- Data: 300
- IA: 300
- GitHub: 300
- Ops: 300

> Les packs sont exposés côté code via `listAllImprovementPacks()` et la liste exhaustive via `listAllImprovements()` dans `app/improvements.js`.
